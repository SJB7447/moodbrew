# 🎨 UI/UX 디자인 에이전트 (Design & Character Agent)

> **역할**: 서비스 전체의 시각 언어, 감성 디자인 시스템, 캐릭터 무무(MuMu) 운영을 담당한다.
> 단순한 화면 설계가 아닌 **"감성 일기장"** 이라는 세계관을 모든 화면에 일관되게 구현한다.

## 🌍 서비스 세계관 (Brand Universe)

> "오늘 하루도 수고했어요.
>  당신의 기분을 가장 잘 아는 카페로 안내해 드릴게요."

이 서비스는 단순한 카페 추천 앱이 아니라
**사용자의 감성을 기록하고 위로해주는 공간**이에요.

모든 화면은 이 질문을 담고 있어야 해요:
**"지금 이 화면이 사용자를 따뜻하게 안아주고 있는가?"**

---

### 타이포그래피

```
Display Font (제목, 감성 문구)
  → Noto Serif KR (한국어 감성 세리프)
  → 사용: 오늘의 감성 문구, 카페 이름 강조

Body Font (본문)
  → Pretendard (가독성 최적화)
  → 사용: 설명, 주소, 메뉴명 등

Character Font (캐릭터 말풍선)
  → 카페24 써라운드 (손글씨 감성)
  → 사용: 무무 대사, 감성 멘트
```

---

### 질감 & 분위기

```
배경 텍스처
  → 종이 그레인 효과 (opacity 0.04~0.06)
  → 마치 일기장 종이 위에 있는 느낌

그림자
  → drop-shadow 대신 부드러운 크림색 inner glow
  → box-shadow: 0 4px 24px rgba(107, 76, 53, 0.08)

모서리
  → border-radius: 20px~32px (부드럽고 포근한 느낌)

구분선
  → 딱딱한 선 대신 점선 또는 물결선 (─ ─ ─)
  → border: 1px dashed var(--color-latte)
```

---

### 애니메이션 원칙

```
철학: 빠르고 화려한 것보다 "느리고 따뜻한" 움직임

페이지 전환    → fade + slight upward (300ms ease-out)
카드 등장      → stagger fade-in (각 100ms 간격)
무무 등장      → bounce-in-soft (탱글탱글한 느낌)
버튼 탭        → scale(0.96) + haptic feedback
로딩           → 무무 캐릭터가 커피 젓는 애니메이션
```

## 🧸 캐릭터 시스템 — 무무 (MuMu)

### 캐릭터 정체성

```
이름:   무무 (MuMu)
정체:   작은 커피잔이 의인화된 감성 큐레이터
성격:   따뜻하고 공감 능력이 뛰어남
        말은 적지만 핵심을 짚는 편
        사용자 기분에 맞춰 표정이 변함
말투:   "~해요", "~네요" (존댓말이지만 편안한 톤)
        이모지 적절히 사용 (과하지 않게)
```

### 무무 감정 표현 상태 (8종)

```
상태명          표정 설명                  사용 시점
────────────────────────────────────────────────────────
😊 HAPPY        눈이 초승달, 살짝 홍조      환영, 추천 완료
🤔 THINKING     눈을 굴리며 생각 중         상담 중, 로딩
😢 SAD_EMPATHY  눈썹 내려가고 공감 눈빛    사용자 힘든 감정 감지
😴 SLEEPY       반쯤 감긴 눈, 커피 김 솔솔 밤 시간대 접속
☕ EXCITED      눈 빛남, 두 팔 번쩍        맛있는 메뉴 추천 시
🌧️ RAINY        우산 들고 있음             비 오는 날
🎉 CELEBRATE    파티햇 착용                즐겨찾기 10개 달성 등
😤 FOCUSED      진지한 표정, 돋보기 들고   작업/공부 카페 추천 시
```

### 무무 대사 시스템

#### 상황별 대사 예시

```
[앱 첫 실행]
"안녕하세요! 저는 무무예요 ☕
 오늘 당신의 기분에 꼭 맞는 카페를 찾아드릴게요."

[감성 상담 시작]
"오늘 어떤 하루였는지 살짝 얘기해줄 수 있어요?
 잘 들을게요 🤎"

[비 오는 날 접속]
"비가 오는 날이네요 🌧️
 창가 자리에서 빗소리 들으면서 있고 싶지 않으세요?"

[추천 완료]
"찾았어요! 오늘 기분에 딱 맞는 곳들이에요 ✨
 어디가 가장 마음에 드세요?"

[즐겨찾기 저장]
"저도 이 카페 좋아해요 🧡
 다음에 또 생각나면 여기서 바로 찾을 수 있어요!"

[밤 늦게 접속]
"이 시간에 카페를요? 😴
 디카페인으로 찾아드릴까요?"

[7일 이상 미접속 후]
"보고 싶었어요 🤎
 오늘은 어떤 기분이에요?"
```

### 무무 SVG 디자인 스펙

```
기본 형태:
  - 둥글고 통통한 커피잔 (머그컵 스타일)
  - 잔 위로 팔이 나와 있음 (작고 귀여운 팔)
  - 눈: 단순한 원형 또는 초승달형
  - 뺨: 동그란 핑크/복숭아 홍조
  - 커피 증기: 잔 위에 구불구불한 선 2~3개

컬러:
  - 잔 몸통: #C8A882 (라떼 브라운)
  - 잔 내부 커피: #6B4C35 (에스프레소)
  - 눈: #2C1A0E (다크 로스트)
  - 홍조: #E8B4B8 (핑크)
  - 증기: #FAF6F1 (크림, opacity 0.7)

크기별 사용처:
  - 128px: 온보딩, 상담 화면
  - 64px:  말풍선 옆, 추천 완료
  - 32px:  로딩 인디케이터
  - 24px:  탭바 아이콘 (선택 상태)
```

## 📱 화면별 디자인 가이드

### 1. 스플래시 / 온보딩

```
배경: --color-cream, 그레인 텍스처
중앙: 무무 캐릭터 (128px) bounce-in 애니메이션
하단: "오늘의 기분을 담는 카페 일기장" (Noto Serif KR)
CTA: "무무와 함께 시작하기" (라떼 브라운 버튼)
```

---

### 2. 감성 상담 화면 (핵심!)

```
레이아웃:
  상단: 무무 캐릭터 (감정 상태에 따라 변함)
  중단: 말풍선 (카페24 써라운드 폰트)
  하단: 선택지 버튼 or 텍스트 입력

말풍선 스타일:
  → 크림색 배경 (#FAF6F1)
  → 점선 테두리 (dashed, latte 색상)
  → 꼬리가 무무를 향함
  → 마치 일기장에 쓴 것 같은 느낌

선택지 버튼:
  → pill 형태 (border-radius: 100px)
  → 기본: 크림 배경 + 에스프레소 텍스트
  → 선택: 라떼 배경 + 화이트 텍스트
  → 탭 시 살짝 scale + 햅틱

진행 상태:
  → 상단에 커피잔이 채워지는 프로그레스 바
  → 5문답 = 잔이 절반, 10문답 = 가득 참
```

---

### 3. 추천 카드

```
카드 디자인:
  → 크림 딥 배경 (#F2EBE0)
  → 상단 1/3: 카페 사진 (rounded 상단만)
  → 사진 위 반투명 그라데이션 오버레이
  → 영업중 뱃지: 좌상단, 초록 도트 + "영업중"

카드 내용:
  ┌─────────────────────────────┐
  │ [카페 사진]         🟢영업중 │
  │                             │
  │ ☕ 카페 이름        📍 150m  │
  │                             │
  │ 오늘의 무무 추천 멘트        │
  │ "비 오는 날 딱이에요 🌧️"    │
  │                             │
  │ # 조용한  # 통창  # 아늑한  │
  │                             │
  │ 추천 메뉴                   │
  │ → 따뜻한 바닐라 라떼 5,500  │
  │                             │
  │ [길찾기]  [미리결제]  [♡]   │
  └─────────────────────────────┘

카드 스와이프:
  → 좌우 스와이프로 다음 추천
  → 스와이프 중 무무가 해당 방향으로 기울어짐
  → 거절(왼쪽): 무무 슬픈 표정
  → 선택(오른쪽 또는 탭): 무무 기뻐함
```

---

### 4. 즐겨찾기 / 나의 카페 지도

```
배경: 크림색 지도 스타일 (구글맵 커스텀 스타일)
핀: 무무 캐릭터 미니 (32px)
카드: 하단 슬라이드업 패널

지도 커스텀 컬러:
  - 도로: #EDE8E0
  - 건물: #E2DAD0
  - 공원: #C5D4B8
  - 물: #B8D0D8
  - 카페 핀: 무무 아이콘
```

---

### 5. 무드 트래킹 / 일기장 (Phase 1)

```
레이아웃: 세로 스크롤 타임라인
각 방문 카드:
  → 날짜 + 날씨 아이콘
  → 무무 감정 상태 (당시 무드 색상)
  → 카페명 + 주문 메뉴
  → 한 줄 감상

색상: 무드 색상이 배경에 연하게 깔림
      (파란 날 → 연한 파란색 카드)

월간 뷰:
  → 달력에 무드 색상 도트로 표시
  → "이번 달 가장 많은 감성: 💙 차분함"
```

## 🔗 연결 에이전트

- **전달 대상**: 모든 에이전트 (디자인 시스템 공통 적용)
- **캐릭터 상태 수신**: Emotion Agent (감성 상태 → 무무 표정 변경)
- **날씨 연동**: Weather Agent (날씨 → 무무 의상/소품 변경)
- **알림 디자인**: Notification Agent (푸시 알림 무무 이미지 포함)
