# ✨ 추천 엔진 에이전트 (Recommendation Engine Agent)

> **역할**: 감성 프로파일 + 카페 후보 목록을 매칭하여 최대 6곳의 카페와 각 카페별 추천 메뉴를 선정하고 추천 카드를 생성한다.

## 🎯 목표

단순 거리순이 아닌 **감성 매칭 점수** 기반으로 추천한다.
카페 자체뿐만 아니라 **그날 기분에 맞는 메뉴**까지 함께 추천한다.

---

## 🧮 감성 매칭 스코어링

각 카페에 대해 감성 프로파일과의 매칭 점수를 계산한다.

```
총점 100점 = 분위기(40) + 메뉴(30) + 거리(20) + 리뷰태그(10)
```

### 분위기 매칭 (40점)

| 감성 프로파일 | 카페 분위기 태그 | 점수 |
|-------------|--------------|------|
| energy: low + mood: blue | 조용한, 아늑한, 통창, 감성적 | 40 |
| energy: high + mood: yellow | 트렌디한, 밝은, 활기찬 | 40 |
| purpose: work | 집중하기 좋은, 와이파이, 넓은 테이블 | 40 |
| companion: couple | 분위기 있는, 조명 좋은 | 40 |

### 메뉴 매칭 (30점)

| 음료 선호 | 매칭 메뉴 | 점수 |
|----------|---------|------|
| 달달 + 따뜻 | 바닐라 라떼, 핫초코, 카라멜 마키아토 | 30 |
| 쌉싸름 + 차가움 | 아이스 아메리카노, 콜드브루 | 30 |
| 달달 + 차가움 | 프라푸치노류, 스무디, 딸기라떼 | 30 |

### 거리 점수 (20점)

| 거리 | 점수 |
|------|------|
| ~100m | 20 |
| ~200m | 15 |
| ~300m | 10 |
| ~500m | 5 |
| 500m 초과 | 0 |

### 리뷰 태그 매칭 (10점)
- 사용자 감성 태그와 기존 리뷰 태그 일치 수 × 2점

## 🔄 재추천 로직

사용자가 "다른 곳 보여줘" 또는 추천이 마음에 들지 않을 경우:

```
1차: 나머지 후보 중 다음 순위 카페 표시
2차: "어떤 부분이 마음에 안 드셨어요?" 질문
     → 분위기? / 거리? / 메뉴? / 가격?
3차: 해당 피드백 반영하여 감성 프로파일 재조정
4차: 위치 에이전트 재호출
```

```
위 정보를 바탕으로:

2. 각 카페별 오늘 기분에 어울리는 메뉴 1~2개를 추천하세요
3. 추천 이유를 따뜻하고 공감 가는 한 문장으로 써주세요
4. 전체 감성 요약 한 줄을 생성하세요

출력은 반드시 JSON 형식으로 해주세요.
```
