# ⭐ 리뷰 에이전트 (Review Agent)

> **역할**: 방문 후 감성 태그 기반 리뷰를 수집하고, 추천 알고리즘 개선 데이터로 활용한다. 별점 대신 무드 태그 방식으로 진행한다.

---

## 📌 에이전트 정보

| 항목 | 내용 |
|------|------|
| **에이전트 ID** | `review-v1` |
| **버전** | v1.0.0 |
| **담당 Phase** | Phase 1부터 |
| **리뷰 방식** | 감성 태그 (별점 없음) |
| **최대 입력 시간** | 30초 이내 완료 목표 |

---

## 🎯 역할 & 책임

- 방문 완료 후 리뷰 요청 (Navigation Agent의 도착 감지 이후)
- 감성 태그 선택형 리뷰 수집
- 한 줄 감상 선택형 수집
- 리뷰 데이터를 추천 알고리즘에 피드백
- 카페 대표 대시보드에 태그 통계 제공

---

## 💬 리뷰 수집 흐름

```
방문 완료 감지 (도착 후 30분)
        │
        ▼
푸시 알림: "카페 어떠셨어요? 30초면 끝나는 감성 리뷰 남겨주세요 😊"
        │
        ▼
STEP 1 — 분위기 태그 선택 (최대 3개)
STEP 2 — 메뉴 만족도 이모지 선택
STEP 3 — 한 줄 감상 선택 (선택 버튼형)
STEP 4 — 다시 올 것 같아요? (예 / 아니오)
```

---

## 🏷️ 분위기 감성 태그 목록

```
공간 관련          분위기 관련         상황 관련
──────────         ──────────          ──────────
#조용한            #감성적인            #혼자오기좋은
#아늑한            #트렌디한            #데이트하기좋은
#넓은              #빈티지한            #공부하기좋은
#통창있는          #밝고활기찬          #대화하기좋은
#주차편한          #아늑한조명          #비오는날생각나는
#애견동반가능       #인스타감성           #재충전하고싶을때
```

## 💭 한 줄 감상 선택 버튼

```
① "다음에 또 오고 싶어요"
② "특별한 날에 오면 좋겠어요"
③"혼자 집중하고 싶을 때 생각날 것 같아요"
④ "친구를 데려오고 싶어요"
⑤ 직접 입력 (선택)
```

## 📊 추천 알고리즘 피드백 루프

```
리뷰 데이터 누적
        │
        ▼
[카페별 태그 빈도 분석]
 예: "○○카페" → #조용한(87%) #통창있는(64%) #혼자오기좋은(71%)
        │
        ▼
[감성 프로파일 매칭 정확도 향상]
 energy: low + mood: blue → #조용한 #아늑한 카페 가중치 ↑
        │
        ▼
[추천 에이전트 스코어링 업데이트]
```

## 🔗 연결 에이전트

- **이전**: Navigation Agent (도착 감지 후 트리거)
- **출력**: Recommendation Agent (알고리즘 피드백), Owner Agent (통계 데이터)
- **저장**: User Data Agent (리뷰 이력)
- **알림**: Notification Agent (리뷰 요청 푸시)
