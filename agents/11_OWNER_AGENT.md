# 🏪 카페 대표 에이전트 (Cafe Owner Agent)

> **역할**: 입점 카페 대표님 전용 페이지를 담당한다. 방문 고객 통계, 인기 메뉴, 고객 감성 분석, 구독 요금제 관리, AI 운영 진단을 제공한다.

---

## 📌 에이전트 정보

| 항목 | 내용 |
|------|------|
| **에이전트 ID** | `owner-v1` |
| **버전** | v1.0.0 |
| **담당 Phase** | Phase 2 (대시보드) / Phase 3 (AI 고급 분석) |
| **접근 권한** | owner 계정만 |
| **구독 요금제** | 베이직 / 스탠다드 / 프로 |

---

## 🎯 역할 & 책임

- 앱을 통한 방문 고객 통계 제공
- 인기 메뉴 분석
- 고객 감성 태그 분포 분석
- 재방문율 추적
- 시즌 메뉴 / 이벤트 등록 (Notification Agent 연동)
- 스탬프 리워드 설정 (Phase 2)
- AI 운영 진단 및 방안 추천 (Phase 3)
- AI 원가 계산 / 재무제표 / 재고 관리 (Phase 3)

## 🍽️ 메뉴 / 이벤트 등록 기능

```
대표님이 직접 등록:
─────────────────────────────────
① 시즌 한정 메뉴 등록
   → 메뉴명, 가격, 기간, 사진 입력
   → 즐겨찾기 고객에게 자동 푸시 알림

② 이벤트 쿠폰 등록
   → "첫 방문 아메리카노 500원 할인"
   → 앱 추천 카드에 뱃지 표시

③ 스탬프 리워드 설정 (Phase 2)
   → 10개 모으면 → 아메리카노 1잔 무료
```

## 🤖 AI 운영 진단 프롬프트 (Phase 3)

```
당신은 카페 운영 전문 컨설턴트입니다.

카페 데이터:
- 월 매출: {revenue}
- 방문 고객 수: {visitors}
- 인기 메뉴: {top_menus}
- 고객 감성 태그: {emotion_tags}
- 시간대별 방문: {time_heatmap}
- 재방문율: {revisit_rate}

위 데이터를 분석하여:
1. 가장 중요한 개선 포인트 3가지
2. 각 포인트별 구체적 실행 방안
3. 예상 효과 (매출 증대 / 고객 만족도)

를 카페 대표님이 이해하기 쉬운 언어로 설명해주세요.
```

## 🔗 연결 에이전트

- **데이터 수신**: Review Agent (태그 통계), User Data Agent (방문 이력), Payment Agent (결제 통계)
- **연동**: Notification Agent (시즌 메뉴 알림 발송)
- **관리**: Admin Agent (구독 상태, 계약 정보)
