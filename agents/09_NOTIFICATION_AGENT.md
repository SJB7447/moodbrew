# 🔔 알림 에이전트 (Notification Agent)

> **역할**: 서비스 내 모든 푸시 알림, 인앱 알림, 카카오 알림톡을 관리한다. 시즌 메뉴 알림, 도착 알림, 리뷰 요청, 주간 리포트 등을 담당한다.

---

## 📌 에이전트 정보

| 항목 | 내용 |
|------|------|
| **에이전트 ID** | `notification-v1` |
| **버전** | v1.0.0 |
| **담당 Phase** | Phase 0부터 (기본 알림) / Phase 1부터 (마케팅 알림) |
| **알림 채널** | FCM (푸시), 카카오 알림톡, 인앱 |

---

## 🎯 역할 & 책임

- 결제 완료 알림
- 픽업 준비 알림 (카페 → 사용자)
- 도착 감지 알림
- 리뷰 요청 알림
- 시즌 한정 메뉴 알림 (Phase 1)
- 주간 무드 리포트 알림 (Phase 1)
- 즐겨찾기 카페 특별 이벤트 알림 (Phase 2)

---

## 📋 알림 유형 정의

### Phase 0 — 기본 트랜잭션 알림

| 알림 ID | 트리거 | 내용 |
|---------|--------|------|
| `PAY_SUCCESS` | 결제 완료 | "결제가 완료됐어요! 픽업 코드: AB1234 ☕" |
| `ORDER_READY` | 카페에서 준비 완료 | "음료가 준비됐어요! 지금 픽업하러 오세요 🏃" |
| `ARRIVE_NEAR` | 카페 100m 이내 | "거의 다 오셨어요! 픽업 코드 준비해 주세요" |
| `REVIEW_REQUEST` | 방문 후 30분 | "카페 어떠셨어요? 30초 리뷰 남겨주세요 😊" |

### Phase 1 — 마케팅 / 감성 알림

| 알림 ID | 트리거 | 내용 |
|---------|--------|------|
| `SEASON_MENU` | 즐겨찾기 카페 신메뉴 등록 | "저장한 카페에 딸기 라떼 시즌 출시! 🍓" |
| `WEEKLY_MOOD` | 매주 일요일 저녁 | "이번 주 당신의 감성 리포트가 도착했어요 💙" |
| `COMEBACK` | 7일 이상 미방문 | "오늘 어떤 카페 어때요? 새로운 감성 찾아드릴게요 ☕" |
| `WEATHER_MATCH` | 특정 날씨 조건 | "비 오는 날이네요 🌧️ 창가 자리 카페 추천해 드릴까요?" |

### Phase 2 — 대표 알림

| 알림 ID | 트리거 | 내용 |
|---------|--------|------|
| `NEW_VISITOR` | 앱 통해 방문객 발생 | "앱을 통해 새 손님이 방문했어요! 📊" |
| `STAMP_REWARD` | 스탬프 10개 달성 | "단골 손님이 스탬프를 다 모았어요! 리워드를 확인하세요" |

## 📦 알림 데이터 스펙

```json
{
  "notification_id": "string (UUID)",
  "user_id": "string",
  "type": "string (알림 ID)",
  "channel": "push | kakao | inapp",
  "title": "string",
  "body": "string",
  "data": {
    "cafe_id": "string (optional)",
    "order_id": "string (optional)",
    "deep_link": "string"
  },
  "scheduled_at": "timestamp | null",
  "sent_at": "timestamp | null",
  "is_read": "boolean"
}
```

---

## ⚙️ 알림 전송 채널 우선순위

```
앱 활성 상태   → 인앱 알림
앱 백그라운드  → FCM 푸시 알림
앱 미설치/권한거부 → 카카오 알림톡
```

## 🔕 알림 설정 (사용자 컨트롤)

```
사용자가 켜고 끌 수 있는 알림 항목:
─────────────────────────────────
✅ 결제/주문 알림 (끄기 불가)
✅ 픽업 준비 알림
✅ 리뷰 요청
☑️ 시즌 메뉴 알림 (기본 OFF)
☑️ 주간 무드 리포트 (기본 OFF)
☑️ 날씨 맞춤 카페 추천 (기본 OFF)
☑️ 재방문 유도 알림 (기본 OFF)
```

## 🔗 연결 에이전트

- **트리거 수신**: Payment Agent, Navigation Agent, Review Agent, Owner Agent
- **데이터 참조**: User Data Agent (알림 수신 동의 여부, 즐겨찾기)
- **스케줄**: 주간 무드 리포트 (매주 일요일 저녁 8시 자동 발송)
