# 🛠️ 관리자 에이전트 (Admin Agent)

> **역할**: 서비스 운영자를 위한 전체 데이터 모니터링, 카페 대표 컨택 CRM, 게스트 현황, 서비스 통계를 관리한다.

---

## 📌 에이전트 정보

| 항목 | 내용 |
|------|------|
| **에이전트 ID** | `admin-v1` |
| **버전** | v1.0.0 |
| **담당 Phase** | Phase 0부터 (기본) / Phase 2부터 (CRM 고도화) |
| **접근 권한** | admin 계정만 |
| **접근 방법** | 별도 관리자 웹 대시보드 |

---

## 🎯 역할 & 책임

- 전체 사용자 현황 모니터링
- 게스트 50명 카운터 관리
- 카페 입점 현황 관리
- 카페 대표 컨택 CRM
- 서비스 전체 KPI 대시보드
- 추천 알고리즘 성능 모니터링
- 결제 통계 및 정산 관리

---

## 📊 관리자 대시보드 구성

### 1. 서비스 현황 (실시간)

```
┌──────────────────────────────────────────────┐
│ 오늘의 서비스 현황                              │
│                                               │
│  👥 총 사용자    게스트    회원     활성        │
│     1,247명     43명     1,204명  287명        │
│                                               │
│  ☕ 추천 횟수   결제 건수   전환율              │
│     892회       234건      26.2%              │
│                                               │
│  💰 오늘 결제액    이번 달                     │
│     1,240,000원   38,700,000원                │
└──────────────────────────────────────────────┘
```

### 2. 게스트 카운터 모니터링

```
게스트 현황
─────────────────────────────
현재 게스트 수: 43 / 50명

[████████████████░░░░] 86%

⚠️ 7명 남음 → 회원 전환 준비 필요
```

### 3. 입점 카페 현황

```
총 입점 카페: 127곳
이번 주 신규: 8곳
활성 영업 중: 89곳
휴업 중: 12곳
탈퇴: 3곳
```

### 4. 추천 성능 KPI

```
추천 → 선택 전환율:  68.3%
추천 → 결제 전환율:  26.2%
재방문율 (30일):     41.7%
평균 매칭 스코어:    73.4점
리뷰 작성률:         54.2%
```

## 🤝 카페 대표 CRM (Phase 2)

카페 대표님들과의 입점 협약 및 관계 관리:

```json
{
  "cafe_crm": {
    "cafe_id": "string",
    "cafe_name": "string",
    "owner_name": "string",
    "contact_phone": "string",
    "contact_email": "string",
    "contract_status": "prospecting | contacted | contracted | active | paused",
    "subscription_plan": "none | basic | standard | pro",
    "monthly_visitors": "number",
    "monthly_revenue_via_app": "number",
    "last_contact_date": "date",
    "memo": "string"
  }
}
```

### CRM 파이프라인

```
발굴        →  1차 컨택   →  미팅     →  계약     →  활성 운영
(앱에 없는     (전화/메일)    (데모)      (구독)      (모니터링)
 인기카페 탐색)
```

## 🔧 관리 기능

- 사용자 계정 정지 / 복구
- 카페 정보 수동 수정
- 리뷰 모더레이션 (부적절한 리뷰 삭제)
- 알림 일괄 발송 (공지사항)
- 게스트 → 회원 전환 이벤트 설정
- 추천 알고리즘 가중치 조정
